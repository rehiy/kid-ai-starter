# 趣味学AI

🤖 **欢迎来到AI冒险乐园！** 🎨

想知道你手机里的助手怎么听懂说话吗？AI为什么能推荐你喜欢的歌曲？这个仓库为你打开一扇窗！

**“趣味学AI”** 是一个给小朋友准备的AI启蒙宝盒。里面没有难懂的代码，只有：

- 🧠 **一个聪明的机器人朋友**：用孩子的方式理解AI。
- 🔍 **七大冒险关卡**：从认识AI、图像识别到AI对话和音乐节奏，玩着学。
- 🎯 **生活中的魔法**：发现手机助手、拍照美颜里的AI秘密。
- ❓ **问答小屋**：解答“AI比人聪明吗？”等有趣问题。

用大约50分钟，开启一段充满色彩和惊喜的人工智能发现之旅吧！适合家长、老师和所有想让孩子们爱上科技的你。
